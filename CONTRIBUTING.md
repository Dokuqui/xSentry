# Contributing to xSentry

Thank you for your interest in contributing! xSentry is a community-driven project.

## Getting Started
- Fork the repo and clone your fork.
- Install Go 1.21+ and run `go mod tidy`.
- Create a branch: `git checkout -b feature/your-feature`.

## Development Workflow
- Build: `go build -o xSentry ./cmd/xSentry`.
- Use pre-commit hook: `./xSentry -install-hook`.

## Submitting Changes
- Push to your fork and open a PR to `main`.
- Include tests for new features.
- Follow Go conventions (e.g., effective-go guidelines).

## Code of Conduct
See [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).

## Questions?
Open an issue or discuss in GitHub Discussions.
