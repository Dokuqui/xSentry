# Security Policy

## Supported Versions
xSentry follows semantic versioning. Only the latest release receives security updates.

## Reporting a Vulnerability
Please do NOT report security issues publicly. Email - doku_isv@icloud.com with details.

- Describe the issue, including steps to reproduce.
- Include affected versions.
- We aim to triage within 48 hours.

## Disclosure
We follow responsible disclosure: acknowledge, fix, then release.
